NOT !

The logical operators AND and OR require TWO operands, of which apply the overall evaluation of the two operands. The logical operator, NOT requires only a single operand. The logical operators are thereafter used within conditional statements, thus resulting in a single outcome.

  3. - * is an unary operator namely: ++ or --

for ( x = val ; condition ; x** )
{
    code
}
