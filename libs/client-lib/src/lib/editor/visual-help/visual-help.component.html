condition

This is the visual representation of how the condtion of the game turn will be defined.

Format

Example

From the above example we deduce the following information:

  • - The condition accepts a parameter , t of type tile.
  • - The State of the board game is then evaluated.
  • - If there is no piece on the tile t, then the variable ans is set to true.
  • - The result is then returned using the return method.

Code Interpretation

condition(t)
{
     consider:this.state.board
     let ans = false
     if (t.pieces.length < 1)
     {
         ans = true
     }
     return ans
}
